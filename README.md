# UWImage-Restoration
To restore the image UW<br>
本项目用于学习水下图像恢复技术，作者为中山大学学生，比赛性质，非商业。欢迎交流。<br>
解决问题：
1.对图像进行统计分析。<br>
2.构建退化模型<br>
3.构建反演复原模型<br>
4.构建复杂环境下的模型<br>
5.比较单一与其他模型给建议<br>
代码结构  
showraw.py用于统计展示源图的各项基本信息<br>
sort.py用于分类图片，并输出至result.csv<br>
judge.py负责几种异常情况判断<br>
push和clone用于克隆和推送至github
